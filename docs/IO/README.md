
# XALG-IO, File-based and CoreData solution for Trees and Graphs

## Text-based format

One graph, one file.

Stores only vertex identifier, edge identifier and edge weight


Learnt from VFL(Visual Formatting Language) of UIKit AutoLayout

### For Graph

```
a1 - b
a2 - b

b - (x) - c 

```
